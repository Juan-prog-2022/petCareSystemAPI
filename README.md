# 🐾 Sistema de Gestión para Veterinarias y Pet-Shops

Este proyecto es una solución integral para la gestión de **veterinarias** y **pet-shops**, desarrollada con **React** en el frontend y **Spring Boot** en el backend. 
Permite gestionar clientes, mascotas, turnos, productos y servicios de forma eficiente y segura.

---

## 🚀 Tecnologías Utilizadas

### 🖥️ Frontend
- React
- TypeScript
- Tailwind CSS
- Axios
- React Router

### 🛠️ Backend
- Java
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- JPA + Hibernate
- PostgreSQL

---

## 🧩 Funcionalidades

- 🐶 Registro y gestión de mascotas
- 👨‍⚕️ Gestión de clientes y turnos veterinarios
- 🛍️ Control de productos y stock
- 🔐 Autenticación segura con JWT
- 👨‍💼 Panel de administración con roles
- 📊 Estadísticas generales
